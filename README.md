# 🛡️ Full Cybersecurity Roadmap & Lab Course

A structured 180-day (6-month) cybersecurity learning platform. This course provides daily tasks, hands-on VM lab exercises, and a built-in validation system to track progress.

## 🚀 Live Demo
**[Click Here to Access the Lab](https://Aravind93911.github.io/your-repo-name/)** 
*(Replace 'your-repo-name' with the actual name of your repository)*

## ✨ Key Features
- **Daily Progression**: 180 curated days of tasks following a logical learning path.
- **Output Validation**: You must submit the correct lab output (Regex/SHA-256) to unlock the next day.
- **Local persistence**: Progress is saved automatically in your browser's `localStorage`.
- **Responsive UI**: Optimized for desktop and tablet studying.
- **VM-Based Labs**: Integrated instructions for Kali Linux and Metasploitable environments.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, Modern JavaScript (ES6+)
- **Security**: SubtleCrypto API for SHA-256 result validation
- **Deployment**: GitHub Pages

## 📖 How to Use
1. **Setup VM**: Follow the instructions in the "VM Lab" section of the website.
2. **Start Day 1**: Read the theory and perform the practical tasks in your VM.
3. **Submit Output**: Paste the requested terminal output or hash into the submission box.
4. **Complete**: Once validated, mark the day complete to unlock Day 2.

## 📁 Project Structure
- `index.html`: Main application interface.
- `style.css`: Professional dashboard styling and animations.
- `script.js`: Core logic for progression, validation, and storage.
- `days.js`: Data file containing the roadmap, tasks, and validation keys.
